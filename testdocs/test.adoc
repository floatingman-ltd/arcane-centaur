== Code Block with Callouts

[source,lua]
----
vim.b.maplocalleader = ","                         -- <1>

vim.keymap.set("n", "<localleader>p", function()   -- <2>
  vim.notify("AsciiDoc preview triggered")
end, { buffer = true })
----
<1> Set localleader to `,` for this buffer only.
<2> Map `,p` as a buffer-local keymap.
