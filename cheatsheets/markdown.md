# Markdown

**LocalLeader** = `,`

---

## Navigation (mkdnflow)

| Key | Action |
|-----|--------|
| `Enter` | Follow link under cursor |
| `Backspace` | Go back to previous file |
| `Tab` | Jump to next link in buffer |
| `S-Tab` | Jump to previous link in buffer |

## Preview

| Key | Action |
|-----|--------|
| `,p` | Toggle browser preview (console: glow, GUI: markdown-preview) |
| `,pp` | Popup preview via glow (always) |
| `,sp` | Open in markserv Docker server (cross-page links) |

## Export

| Key | Action |
|-----|--------|
| `,dp` | Export to PDF with PlantUML diagrams rendered |

---

## MARP Presentations

| Key | Action |
|-----|--------|
| `,mp` | Open slide in preview server |
| `,mx` | Export to PowerPoint (`.pptx`) |
| `,mh` | Export to HTML |
| `,md` | Export to PDF |
