# Haskell

**Leader** = `Space`

---

## haskell-tools — GHCi REPL

| Key | Action |
|-----|--------|
| `<leader>rr` | Toggle GHCi REPL for package |
| `<leader>rf` | Toggle GHCi REPL for current file |
| `<leader>rq` | Quit GHCi REPL |

## haskell-tools — Code Lens & Search

| Key | Action |
|-----|--------|
| `Space-cl` | Run code lenses |
| `Space-hs` | Hoogle search for type under cursor |
| `Space-ea` | Evaluate all code snippets |
