## iron.nvim — REPL

| Key | Action |
|-----|--------|
| `,sl` | Send current line to REPL |
| `,sc` | Send motion or selection to REPL |
| `,sp` | Send current paragraph to REPL |
| `,sf` | Send entire file to REPL |
| `,s<CR>` | Send carriage return to REPL |
| `,si` | Interrupt REPL |
| `,sq` | Quit / exit REPL |
| `,cl` | Clear REPL output |
