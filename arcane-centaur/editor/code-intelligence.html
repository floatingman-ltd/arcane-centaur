LSP

These keymaps are active in any buffer where an LSP server has attached (F#, C#, Markdown, Haskell, Janet, Lua).

Keys Mode Action

gd

Normal

Go to definition

K

Normal

Hover documentation

gr

Normal

Find all references

<leader>rn

Normal

Rename symbol

<leader>ca

Normal

Code action

<leader>e

Normal

Show diagnostic float

[d

Normal

Jump to previous diagnostic

]d

Normal

Jump to next diagnostic

Markdown

sudo apt install marksman

janet_lsp

Janet

jpm install janet-lsp

haskell-language-server

Haskell

ghcup install hls (managed by haskell-tools.nvim)

lua_ls

Lua

See Lua guide

Keymaps defined in lua/config/lsp.lua.

Auto-Completion (nvim-cmp)

Completion Menu

Keys Mode Action

Alt-Space

Insert

Open completion menu on demand

Ctrl-n

Insert (menu open)

Select next suggestion

Ctrl-p

Insert (menu open)

Select previous suggestion

Enter

Insert

Confirm highlighted suggestion

Ctrl-e

Insert

Dismiss completion menu

Ctrl-f

Insert

Scroll documentation preview down

Ctrl-b

Insert

Scroll documentation preview up

Enter only inserts when an entry is actively highlighted — it will not accidentally confirm the first suggestion.

Format-on-Save

Format-on-save is enabled automatically for the following filetypes:

Filetype Formatter

Lisp

LSP (if attached)

Clojure

LSP

LSP

Fennel

LSP

Janet

LSP

F#

LSP (fsautocomplete)

Lua

stylua
