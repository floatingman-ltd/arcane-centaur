Lisp Cheatsheet (Conjure + vim-sexp)

LocalLeader = , in Lisp / Clojure / Scheme / Fennel buffers

→ Back to main cheatsheet · Full guide: ../guides/lisp.md · Janet-specific: janet.md

Common Lisp REPL Containers

Compose file: docker/lisp-swank/docker-compose.yml

The :4005 contract: whoever owns 127.0.0.1:4005 is the active Lisp. Switch by bringing down the current profile and bringing up the next; reconnect with ,cc.

Implementation Start command

SBCL (default)

LISP_DIR=$PWD docker compose -f ~/.config/nvim/docker/lisp-swank/docker-compose.yml up -d --wait

CCL

LISP_DIR=$PWD docker compose -f ~/.config/nvim/docker/lisp-swank/docker-compose.yml --profile ccl up -d --wait

ECL

LISP_DIR=$PWD docker compose -f ~/.config/nvim/docker/lisp-swank/docker-compose.yml --profile ecl up -d --wait

ABCL

LISP_DIR=$PWD docker compose -f ~/.config/nvim/docker/lisp-swank/docker-compose.yml --profile abcl up -d --wait

 
# Switch: bring down current, bring up next, reconnect
docker compose -f ~/.config/nvim/docker/lisp-swank/docker-compose.yml down
# ...then the start command for the target impl above
# Then in Neovim: ,cc

Other Active Plugins

  • nvim-parinfer — keeps parentheses balanced as you edit indentation (no keymaps)

  • rainbow-delimiters.nvim — color-codes matching delimiters by depth (no keymaps)

  • LSP (none configured by default for Common Lisp) — see lsp.md

  • Formatting (conform.nvim, format-on-save) — see formatting.md

Conjure prefix configured in lua/plugins/lisp.lua. vim-sexp keymaps from tpope/vim-sexp-mappings-for-regular-people. LocalLeader set in after/ftplugin/lisp.lua, clojure.lua, and scheme.lua.
