REPL — iron.nvim

REPL backend: csharprepl for C# buffers · dotnet fsi for F# buffers.

Keys Action

,sl

Send current line to REPL

,sc

Send motion or visual selection to REPL

,sp

Send current paragraph to REPL

,sf

Send entire file to REPL

,s<CR>

Send carriage return to REPL

,si

Interrupt REPL

,sq

Quit / exit REPL

,cl

Clear REPL output

Debugging (nvim-dap + netcoredbg)

Prerequisites: netcoredbg (GitHub-release binary — not dotnet tool), the EasyDotnet server tool (dotnet tool install -g EasyDotnet), and fzf. Details in the .NET guide.
