Janet Cheatsheet (Conjure + vim-sexp)

LocalLeader = , in Janet buffers

→ Back to main cheatsheet · Full guide

This config uses the Janet stdio client (janet -n -s). Conjure spawns it automatically on first evaluation; use ,cs/,cS to restart it manually.

Normal

Conjure — Log Window

Keys Action

,ls

Open log in horizontal split

,lv

Open log in vertical split

,lt

Open log in new tab

,lg

Toggle log window open / closed

,ll

Jump to latest result in log

,lq

Close all visible log windows

Conjure — Connection (Janet stdio)

Keys Action

,cs

Start the Janet REPL subprocess

,cS

Stop the Janet REPL subprocess

Other Active Plugins

  • nvim-parinfer — keeps parentheses balanced as you edit indentation (no keymaps)

  • rainbow-delimiters.nvim — color-codes matching delimiters by depth (no keymaps)

  • LSP (janet_lsp) — see lsp.md

  • Formatting (conform.nvim, format-on-save) — see formatting.md

Conjure prefix configured in lua/plugins/lisp.lua. vim-sexp keymaps from tpope/vim-sexp-mappings-for-regular-people. LocalLeader set in after/ftplugin/janet.lua.
