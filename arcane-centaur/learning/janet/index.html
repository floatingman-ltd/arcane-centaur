Janet Learning Series

Interactive, REPL-driven lessons for Janet in this Neovim configuration. Work through each lesson with Neovim open and a Conjure REPL running alongside.

Core Lessons

# Lesson Topic

01

Setup

Install Janet, jpm, LSP — verify all plugins are working

02

First Steps

Syntax, data types, control flow, and the Neovim toolchain hands-on

03

Functions in Depth

Closures, higher-order functions, threading macros (->, →>)

04

Sequences

seq macro, generators, lazy iteration, destructuring

05

Modules

import, use, jpm project layout, the spork library

06

Error Handling

try/catch, protect, error, fibers as an error mechanism

07

Macros

Quasiquoting, defmacro, when to use macros vs functions

How to Use This Series

Each lesson is self-contained and REPL-driven. Open a scratch file and the Conjure log before starting any lesson:

nvim scratch.janet
,lv

Evaluate every code block as you read — use ,ee on the form under the cursor or ,eb for the whole buffer. Results appear in the Conjure log pane in real time.
