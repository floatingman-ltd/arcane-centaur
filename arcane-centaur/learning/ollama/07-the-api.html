07 — The Ollama API

Avante is a Neovim interface over the Ollama HTTP API. This lesson shows you what the API looks like directly, so you understand what’s happening under the hood and can use it for scripting or exploration outside of Neovim.

Make sure Ollama is running:

curl -s http://127.0.0.1:11434/

Confirm you see: Ollama is running

1. The Two Core Endpoints

Ollama exposes a REST API on port 11434. The two endpoints you’ll use most:

Endpoint Purpose

POST /api/generate

Single-turn: send a prompt, get a response. No conversation history.

POST /api/chat

Multi-turn: send a list of messages (roles: system/user/assistant), get a response. This is what Avante uses.

2. /api/generate — Simple Prompts

curl -s http://127.0.0.1:11434/api/generate \
  -d '{
    "model": "llama3.1:8b",
    "prompt": "Reply with exactly: API IS WORKING",
    "stream": false
  }'

Confirm you see: a JSON response with a response field containing API IS WORKING.

The key fields in the response:

{
  "model": "llama3.1:8b",
  "response": "API IS WORKING",
  "done": true,
  "total_duration": 1234567890,
  "eval_count": 5
}

Try it: change the prompt to ask something you’re curious about. Extract just the response:

curl -s http://127.0.0.1:11434/api/generate \
  -d '{"model": "llama3.1:8b", "prompt": "Name three functional languages.", "stream": false}' \
  | python3 -c "import sys, json; print(json.load(sys.stdin)['response'])"

Confirm you see: three language names with no surrounding JSON.

3. /api/chat — Multi-Turn Conversations

This is what Avante uses. Messages have roles: system, user, assistant.

curl -s http://127.0.0.1:11434/api/chat \
  -d '{
    "model": "llama3.1:8b",
    "stream": false,
    "messages": [
      {
        "role": "system",
        "content": "You are a terse Neovim expert. One sentence answers only."
      },
      {
        "role": "user",
        "content": "What does <leader> mean in Neovim?"
      }
    ]
  }'

Confirm you see: a JSON response with a message.content field containing a one-sentence answer.

To continue the conversation, append the assistant’s response and your next message to the messages array. This is exactly what Avante manages for you automatically.

4. Streaming Responses

By default ("stream": false) Ollama waits until the response is complete before returning. With streaming ("stream": true), it returns a series of newline-delimited JSON objects as the response is generated — one per token.

curl -s http://127.0.0.1:11434/api/generate \
  -d '{"model": "llama3.1:8b", "prompt": "Count from one to five.", "stream": true}'

Confirm you see: a stream of JSON objects, each with a partial response field, followed by a final object with "done": true.

Streaming is how Avante renders text progressively in the buffer as the model generates it.

5. Listing Models via the API

curl -s http://127.0.0.1:11434/api/tags | python3 -m json.tool

Confirm you see: a JSON object listing your pulled models with names, sizes, and digests.

This is equivalent to ollama list inside the container.

6. What Avante Is Sending

When you submit a message to Avante with <leader>ao, it:

  1. Collects your conversation history (all previous user + assistant messages)

  2. Prepends the system prompt if one is active

  3. Sends a POST /api/chat request with "stream": true

  4. Renders each token as it arrives in the Avante buffer

The only difference between using Avante and using curl directly is that Avante manages the conversation history, handles streaming rendering, and integrates with the editor (visual selection context, buffer awareness in future versions).

7. A Simple Shell Script

Now that you know the API, you can use Ollama from outside Neovim:

#!/usr/bin/env bash
# ask-ollama: quick command-line query to the local Ollama service
# Usage: ask-ollama "What does xdg-open do?"

PROMPT="$1"
MODEL="${OLLAMA_MODEL:-llama3.1:8b}"

curl -s http://127.0.0.1:11434/api/generate \
  -d "{\"model\": \"$MODEL\", \"prompt\": \"$PROMPT\", \"stream\": false}" \
  | python3 -c "import sys, json; print(json.load(sys.stdin)['response'])"

Save this as ~/.local/bin/ask-ollama, make it executable (chmod +x), and use it from any terminal.

Try it:

ask-ollama "What is the Neovim filetype for Janet source files?"
