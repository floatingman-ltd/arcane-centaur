Ollama Learning Path

Interactive lessons for using Ollama with this Neovim configuration. Work through each lesson with your editor open and the Ollama service running.

Core Lessons

# Lesson Topic

01

Setup & Verification

Start the Docker service, pull a model, confirm Avante connects end-to-end

02

First Conversations

Avante buffer layout, sending messages, conversation history, switching providers

03

Prompting Fundamentals

Why prompt quality matters, specificity, context, role-setting

04

Code Assistance Workflow

Visual selection → Avante: explain, review, and refactor code in this config

05

Model Selection

RAM/speed/quality tradeoffs, pulling models, switching the active model

06

System Prompts & Context

Shaping model behaviour, context window limits, multi-turn conversations

07

The Ollama API

HTTP API, curl exercises, what Avante sends under the hood
