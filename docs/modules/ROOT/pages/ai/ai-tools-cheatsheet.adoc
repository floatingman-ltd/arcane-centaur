== AI Tools Cheat Sheet

Covers in-editor keymaps for the *Claude CLI* (`claude`) and the
*OpenSpec* workflow tool.

*Leader* = `Space`

'''''

== Claude CLI

These commands send the current visual selection (or whole buffer when
no selection is active) to the `claude` CLI and display the response in
a floating window.

[width="100%",cols="34%,33%,33%",options="header",]
|===
|Keys |Mode |Action
|`++<++leader++>++gcs` |Normal / Visual |Ask Claude to suggest a shell
command

|`++<++leader++>++gce` |Normal / Visual |Ask Claude to explain the
selected code
|===

=== Commands

[width="100%",cols="50%,50%",options="header",]
|===
|Command |Description
|`:ClaudeSuggest` |Send buffer/selection to `claude` for a shell
command suggestion

|`:ClaudeExplain` |Send buffer/selection to `claude` for a code
explanation
|===

=== Floating window controls

[cols=",",options="header",]
|===
|Key |Action
|`q` |Close result window
|`++<++Esc++>++` |Close result window
|===

'''''

== Research Popup

Ask a free-form question from any buffer and read the answer in a floating window.

[width="100%",cols="34%,33%,33%",options="header",]
|===
|Keys |Mode |Action
|`++<++leader++>++?l` |Normal |`:ResearchLocal` — question grounded in this config's keymaps and cheatsheets

|`++<++leader++>++?a` |Normal |`:ResearchAsk` — general-knowledge question (no local context)
|===

=== Commands

[width="100%",cols="50%,50%",options="header",]
|===
|Command |Description
|`:ResearchLocal` |Ask about this Neovim config; answer uses only local context

|`:ResearchAsk` |Ask any general question; no local context prepended
|===

Uses Claude Code's built-in auth (no `ANTHROPIC_API_KEY`). Press `q` or `++<++Esc++>++` to close the result float.

'''''

== OpenSpec

Drive the OpenSpec change workflow from within Neovim. Output is shown
in a bottom split (close with `q` or `++<++Esc++>++`).

[cols=",,",options="header",]
|===
|Keys |Mode |Action
|`++<++leader++>++osn` |Normal |Create a new change (`OpenspecNew`)
|`++<++leader++>++oss` |Normal |Show status (`OpenspecStatus`)
|`++<++leader++>++osl` |Normal |List all changes (`OpenspecList`)
|===

=== Commands

[width="100%",cols="50%,50%",options="header",]
|===
|Command |Description
|`:OpenspecNew ++[++name++]++` |Prompt for a change name (or use
argument) and create it

|`:OpenspecStatus ++[++name++]++` |Show status for all changes, or a
specific `++[++name++]++`

|`:OpenspecList` |List all changes in the repo
|===

The same workflows are available as Claude Code skills:
`/openspec-propose`, `/openspec-apply-change`,
`/openspec-continue-change`, `/openspec-verify-change`,
`/openspec-archive-change`.

== Avante AI Research

[cols=",",options="header",]
|===
|Key |Action
|`++<++leader++>++aa` |Open avante with the current provider (ollama)
|`++<++leader++>++ao` |Select the ollama provider and open avante
|`++<++C++>++s` |Submit the prompt (inside the avante input; `++<++CR++>++` = newline)
|===

_Avante is Ollama-only; the Claude backend is disabled (Anthropic OAuth ToS) — see the guide to re-enable via API key._

_Plugin configured in `lua/plugins/avante.lua`. See xref:ai/ai-tools.adoc#avante[full Avante guide]._
