== Neovim Keybinding Cheat Sheet

*Leader* = `Space` ┬Ę *LocalLeader* = `,` (in all language buffers)

NOTE: Press `*<leader>?*` inside Neovim for an instant, context-aware keybinding reference ŌĆö shows universal bindings plus language-specific bindings for the current filetype. Works in any terminal, including headless SSH sessions.

[cols=",",options="header",]
|===
|Keys |Action
|`++>++)` |Slurp forward ŌĆö pull next element into form
|`++<++)` |Barf forward ŌĆö push last element out of form

|`++<++f` |Move current form left among siblings

|`++<++e` |Move current element left

|`cse++[++` or `cse++]++` |Surround element with `++[]++`
