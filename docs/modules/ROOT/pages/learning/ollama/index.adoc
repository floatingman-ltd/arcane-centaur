= Ollama Learning Path

Interactive lessons for using Ollama with this Neovim configuration.
Work through each lesson with your editor open and the Ollama service running.

== Core Lessons

[cols="^1,3,8",options="header"]
|===
| # | Lesson | Topic

| 01
| xref:learning/ollama/01-setup.adoc[Setup & Verification]
| Start the Docker service, pull a model, confirm Avante connects end-to-end

| 02
| xref:learning/ollama/02-first-conversations.adoc[First Conversations]
| Avante buffer layout, sending messages, conversation history, switching providers

| 03
| xref:learning/ollama/03-prompting-fundamentals.adoc[Prompting Fundamentals]
| Why prompt quality matters, specificity, context, role-setting

| 04
| xref:learning/ollama/04-code-assistance.adoc[Code Assistance Workflow]
| Visual selection → Avante: explain, review, and refactor code in this config

| 05
| xref:learning/ollama/05-model-selection.adoc[Model Selection]
| RAM/speed/quality tradeoffs, pulling models, switching the active model

| 06
| xref:learning/ollama/06-system-prompts.adoc[System Prompts & Context]
| Shaping model behaviour, context window limits, multi-turn conversations

| 07
| xref:learning/ollama/07-the-api.adoc[The Ollama API]
| HTTP API, `curl` exercises, what Avante sends under the hood
|===
