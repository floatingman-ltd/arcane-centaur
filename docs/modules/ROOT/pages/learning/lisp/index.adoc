= Common Lisp Learning Series

Interactive, REPL-driven lessons for Common Lisp in this Neovim configuration.
Work through each lesson with Neovim open and a Conjure REPL running alongside.
All code blocks are evaluable with `,ee` (form under cursor) or `,eb` (whole buffer).

No LSP is configured for Common Lisp by design — the running image *is* the
language server. Lesson 02 teaches you to get arglists, documentation, and
source location from the live Swank connection.

== How to Use This Series

Start by launching the SBCL container and opening a scratch file:

[source,sh]
----
LISP_DIR=$PWD docker compose -f ~/.config/nvim/docker/lisp-swank/docker-compose.yml up -d --wait
nvim scratch.lisp
----

Open the Conjure log immediately:

....
,lv
....

Evaluate every code block as you read — use `,ee` on the form under the cursor
or `,eb` for the whole buffer.
Results appear in the Conjure log pane in real time.

To switch to a different implementation at any point, see
xref:getting-started.adoc#sbcl-swank[Getting Started: Common Lisp REPL].
