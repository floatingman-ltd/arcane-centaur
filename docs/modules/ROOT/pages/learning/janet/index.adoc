= Janet Learning Series

Interactive, REPL-driven lessons for Janet in this Neovim configuration.
Work through each lesson with Neovim open and a Conjure REPL running alongside.

== How to Use This Series

Each lesson is self-contained and REPL-driven.
Open a scratch file and the Conjure log before starting any lesson:

[source,sh]
----
nvim scratch.janet
----

....
,lv
....

Evaluate every code block as you read — use `,ee` on the form under the cursor or `,eb` for the whole buffer.
Results appear in the Conjure log pane in real time.
