== Neovim Keybinding Cheat Sheet

*Leader* = `Space` · *LocalLeader* = `,` (in all language buffers)

NOTE: Press `*<leader>?*` inside Neovim for an instant, context-aware keybinding reference — shows universal bindings plus language-specific bindings for the current filetype. Works in any terminal, including headless SSH sessions.

[cols=",,",options="header",]
|===
|Keys |Mode |Action

|`++<++leader++>++Y` |Normal |Yank line to system clipboard

|`++<++leader++>++d` |Normal / Visual |Cut to system clipboard

=== Structural Editing — vim-sexp

[cols=",",options="header",]
|===
|Keys |Action
|`++>++)` |Slurp forward — pull next element into form
|`++<++)` |Barf forward — push last element out of form
|`++<++(` |Slurp backward
|`++>++(` |Barf backward
|`++>++f` |Move current form right among siblings
|`++<++f` |Move current form left among siblings
|`++>++e` |Move current element right
|`++<++e` |Move current element left
|`cse(` or `cse)` |Surround element with `()`
|`cse++[++` or `cse++]++` |Surround element with `++[]++`
|`dsf` |Delete surrounding function call (splice)
|===

→ xref:languages/lisp-cheatsheet.adoc[Full reference]
