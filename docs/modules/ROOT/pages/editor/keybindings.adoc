== Neovim Keybinding Cheat Sheet

*Leader* = `Space` · *LocalLeader* = `,` (in all language buffers)

TIP: Press any prefix key (e.g. `Space`, `Space g`, `Space o`) and *pause* — **which-key** will pop up a menu showing all available keymaps and their descriptions. Keep typing to dismiss it instantly.

NOTE: Press `*<leader>?*` inside Neovim for an instant, context-aware keybinding reference — shows universal bindings plus language-specific bindings for the current filetype. Works in any terminal, including headless SSH sessions.

|===
|Keys |Action
