# Magical NeoVIM

A Neovim configuration for the discerning coder — featuring REPL-driven development, AI assistance, and rich documentation tooling.

📖 **[Full documentation site →](https://floatingman-ltd.github.io/arcane-centaur)**

## Quick Install

```sh
git clone https://github.com/floatingman-ltd/arcane-centaur ~/.config/nvim
```

Requires **Neovim ≥ 0.12**. On first launch, [lazy.nvim](https://github.com/folke/lazy.nvim) bootstraps itself and installs all plugins automatically.

Full setup instructions (Docker, LSP servers, REPLs, runtimes) are in the **[Getting Started guide](https://floatingman-ltd.github.io/arcane-centaur/guides/getting-started.html)**.
